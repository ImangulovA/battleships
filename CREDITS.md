# Credits

## The puzzle

**Battleships** (also known as *Bimaru*, *Solitaire Battleships*, or *Yubotu*) is
a classic logic puzzle: find a hidden fleet of straight, non-touching ships from
row/column segment counts and a known fleet. This project is an **independent
implementation**: every puzzle here is **generated by our own code**
(`scripts/gen_days.py`) with a constraint-propagation solver that guarantees a
unique, no-guessing solution, cross-checked by an independent backtracking solver.
No third-party puzzle content is bundled or republished.

- The excellent free print puzzles by **Jim Bumgardner / Krazydad**
  (krazydad.com) were used *only as a difficulty reference* while calibrating grid
  sizes and fleets per tier. Those reference files live in `sources/` locally and
  are **git-ignored** — they are not part of this repository or the deployed site.

## The platform

Built on a small reusable "one puzzle a day" web-game template
(SvelteKit + adapter-static, optional Cloudflare Worker for global stats),
shared with the sibling *Daily Train Tracks* project.
